ΜΕΤΑΒΛΗΤΕΣ
    Η μεταβλητή είναι ένα όνομα που αναφέρεται σε μια τιμή. 
    Μπορούμε να την ορίσουμε ως εξής:
    x = 5
    y = 10
    this_is_a_really_long_variable_name = 15

    Οι μεταβλητές μπορούν να πάρουν και να αλλάξουν τιμές ή να παρουν τιμές από
    αποτελέσματα πράξεων ή συναρτήσεων.
    Πχ:
    x = 5
    y = 10
    z = x + y
    Η μεταβλητή z θα πάρει την τιμή 15

ΠΡΑΞΕΙΣ
    Μπορούμε να κάνουμε πράξεις μεταβλητών μεταξύ τους
    Πρόσθεση: x + y
    Αφαίρεση: x - y
    Πολλαπλασιασμός: x * y
    Ακεραια Διαίρεση: x // y (αποτέλεσμα ακέραιος)
    Πραγματική Διαίρεση: x / y (μπορεί να επιστρέψει δεκαδικό αριθμό)
    Υπόλοιπο της διαίρεσης x δια y: x % y
    Δύναμη: x ** y


ΣΥΓΚΡΊΣΕΙΣ
    Μικρότερο: x < y
    Μεγαλύτερο: x > y
    Ίσο: x == y
    Μικρότερο ή ίσο: x <= y
    Μεγαλύτερο ή ίσο: x >= y
    Άνισο: x != y
Οι συγκρίσεις έχουν αποτέλεσμα μια από τις 2 τιμές True ή False

ΣΥΝΘΗΚΕΣ
Έστω ότι θέλουμε αν μια μεταβλητή x είναι θετική να προσθέσουμε 1
αλλιώς να αφαιρέσουμε ένα. Τότε χρησιμοποιούμε το if else

if x > 0:
    x = x + 1
else:
    x = x - 1

Παρατηρούμε ότι στο τέλος του if και του else βάζουμε άνω και κάτω τελία :
Επίσης όσο γράφουμε τις εντολές μέσα στο if ή το else χρειαζόμαστε στοίχιση
πατώντας το κουμπί TAB (πάνω από το CAPS LOCK)

Μπορούμε να έχουμε οτιδήποτε μέσα στο if, ακόμα και άλλο ένα if
Δεν χρειάζεται απαραίτητα να έχουμε το else

if x > 0:
    if x < 10:
        print("hi")
else:
    print("not hi")

ΕΠΑΝΑΛΗΨΕΙΣ

Υπάρχουν 2 βασικοί τρόποι να κάνουμε επανάληψη την python. Το for και το while
Το for μπορούμε να το χρησιμοποιήσουμε όταν ξέρουμε τον αριθμό επαναλήψεων που
θέλουμε να κάνουμε

Πχ 5
for _ in range(5):
    x = x + 1
    print(x)
print("Αυτή η εντολή είναι έξω από το for και θα εκτελεστεί μόνο στο τέλος")

ή n φορές αν το n είναι ακέραιος

for i in range(n):
    print(i)
    print("Το i θα αρχίσει από την τιμή 0 και θα αυξάνει κατά 1 σε κάθε επανάληψη μέχρι να πάρει την τιμή n-1")

Το while εκτελείται όσο μια συνθήκη είναι αληθής, δηλαδή True

while 1 > 0:
    print("Αυτό το while θα τρέχει για πάντα)

x = 0
while x < 5:
    x = x + 1
Πόσες φορές θα τρέξει αυτό;

Έστω ότι θέλω να δω ποια είναι η μεγαλύτερη δύναμη του 2 που είναι μικρότερη από το 1000

power = 0
while 2 ** power < 1000:
    power = power + 1
print("Αφού βγήκα από το while πλέον δεν ισχύει, άρα πρέπει να αφαιρέσω 1")
power = power - 1
